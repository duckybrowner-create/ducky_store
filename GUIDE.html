üìñ Panduan Langkah Demi Langkah

Langkah 1: Membuka Landing Page

1 Buka file landing.html di browser

Anda akan melihat halaman utama DuckyStore dengan fitur-fitur aplikasi.

Langkah 2: Login ke Admin Panel

2 Klik tombol "Admin Login" di halaman landing atau buka login.html
3 Masukkan email dan password (gunakan demo login di atas)
4 Klik tombol "Login"

Langkah 3: Menggunakan Dashboard

Setelah login, Anda akan masuk ke dashboard dengan menu utama:

  • Dashboard - Lihat statistik penjualan
  • Kelola Produk - Tambah, edit, atau hapus produk
  • Statistik - Analisis penjualan detail
  • Profile - Lihat data profil dan ubah password
  • Pengaturan - Konfigurasi aplikasi
  • Logout - Keluar dari akun

‚ùì Troubleshooting

Lupa Password?

Ganti password di menu "Profile" pada dashboard admin. Anda perlu password lama untuk mengubahnya.

Data Hilang?

Data disimpan di localStorage browser. Jika Anda menghapus browser cache, semua data akan hilang. Backup produk Anda secara berkala.

Keranjang Kosong di Halaman Lain?

Setiap kali membuka halaman toko, pastikan halaman sudah fully loaded sebelum menambah produk ke keranjang.
